## Requirements
- PHP 8+
- MySQL / MariaDB
- Local server (XAMPP/WAMP/MAMP or php -S)

## Setup
1. Create a database (e.g. `dev_test`).
2. Import `schema.sql` into that database.
3. Update DB credentials in `api/db.php`.
4. Start a local server in this folder:
   - Option A: Put this folder under your web root (e.g. XAMPP `htdocs`)
   - Option B: `php -S localhost:8000` (run in this folder)
5. Open `index.html` (or http://localhost:8000/index.html).
